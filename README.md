# Évora LeadFlow — Demo Estática Vercel

Esta versão é uma demo premium 100% estática, sem FastAPI e sem função serverless.
Ela foi criada para eliminar o erro `FUNCTION_INVOCATION_FAILED` na fase de apresentação.

## Deploy na Vercel

1. Suba estes arquivos para um repositório GitHub.
2. Importe o repositório na Vercel.
3. Framework Preset: Other.
4. Build Command: deixe vazio.
5. Output Directory: deixe vazio.
6. Não configure variáveis de ambiente.
7. Deploy.

## Telas

- `/`
- `/leads`
- `/vitoria`
- `/materiais`
- `/relatorios`
- `/corretores`
- `/empreendimentos`
- `/configuracoes`

A Vitória é simulada em JavaScript local, sem chamadas a API.
